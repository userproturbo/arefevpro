Описание проекта: Личный сайт CRAZYLIFE

📌 Общая архитектура
Мой личный сайт состоит из пяти основных разделов:
Мои проекты
Фото
Видео
Музыка
Блог
На главной странице каждый раздел представлен в виде крупной карточки по центру экрана.
Дизайн сейчас работает, навигация функционирует корректно.

📁 Контент и функциональность
Каждый раздел открывается в собственную страницу, где отображаются публикации этого раздела.
Внутри публикации доступно:
просмотр контента (текст, фото, видео, музыка),
лайки,
комментарии,
счётчики активности.
Вся функциональность уже связана между собой и работает.

🛠 Админ-панель
В проект встроена админка, где я могу:
создавать публикации,
редактировать,
удалять,
загружать файловый контент.
Админка полностью рабочая и связана с backend + базой данных.

🚀 Ближайшие задачи (перед деплоем)
В ближайшем будущем планируется:

Перенести сайт на VPS
— backend, frontend, бот будут работать на одном российском сервере.

Подключить облачное хранилище (Яндекс Object Storage)
— хранение фото, видео, музыки, больших файлов.

🧩 Что нужно доработать
Чтобы проект стал завершённым и production-ready, необходимо:
1. UI/UX проработка
привести все разделы к единому стилю,
создать ленту Блог ( нужно сделать ввиде ленты на главной странице)
остальные карточки(секции) Мои проекты, Фото, Видео, Музыка нужно подумать как лучше реализовать на странице,
улучшить визуал главной страницы,
продумать поведение элементов при наведении,
выровнять типографику: шрифты, размеры, отступы.

3. Адаптивная верстка
Сайт должен корректно работать на:
мобильных,
планшетах,
ноутбуках,
широких мониторах.
Разрешения должны быть оптимизированы под Tailwind breakpoints.

3. Рефакторинг кода
вынести повторяющиеся компоненты,
оптимизировать API,
улучшить структуру файлов,
оптимизировать загрузку медиа (lazy loading),
упорядочить стили.

4. Главная лента
Обязательная будущая функция:
Когда я публикую пост в блоге — он должен появляться в ленте на главной странице.
Это создаст ощущение живого сайта, как в Instagram/Twitter.
Лента должна содержать:
миниатюру или обложку,
заголовок,
дату,
краткий текст или превью,
лайки + количество комментариев,
пост в ленте может содержать текст,музыку,видео,фото.
